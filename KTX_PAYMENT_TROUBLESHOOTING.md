# KTX 자동결제 실패(결제정보 저장 안 됨) 점검 가이드

아래 알림(`결제 정보가 없어 자동 결제를 진행하지 못했습니다`)이 뜨는 경우, 대부분 **결제수단 등록은 했지만 자동결제에 필요한 동의/인증 상태가 미완료**인 경우가 많습니다.

## 1) 결제수단 등록 직후 재로그인/앱 재시작
- 코레일톡에서 카드 등록 후 즉시 예약을 시도하면, 세션에 반영되지 않아 `결제정보 없음`으로 판단되는 경우가 있습니다.
- 앱을 완전히 종료 후 재실행하고 다시 로그인해 확인하세요.

## 2) 카드 저장과 자동결제 동의는 별개인지 확인
- `카드 저장`만 완료되고, `자동결제(간편결제) 사용 동의`가 미완료면 자동결제가 동작하지 않습니다.
- 결제관리 메뉴에서 아래 항목이 모두 `ON`인지 확인하세요.
  - 기본 결제수단 지정
  - 자동결제 사용 동의
  - 약관/본인인증 완료 상태

## 3) 본인인증(3DS/ISP/앱카드) 미완료 여부 확인
- 일부 카드사는 최초 1회 인증(앱카드, ARS, 3DS)을 마쳐야 자동결제가 가능합니다.
- 카드 재등록 후 **소액 결제 1회 수동 성공**을 먼저 진행하면 이후 자동결제가 되는 케이스가 있습니다.

## 4) 법인/체크/해외카드 제한 확인
- 카드 종류(법인/해외/체크)에 따라 자동결제 지원이 제한될 수 있습니다.
- 가능하면 개인 신용카드로 재등록 후 테스트하세요.

## 5) 예약 시간 임박/잔여시간 부족
- 자동결제 재시도 중 결제 유효시간(예: 10분)이 만료되면 실패합니다.
- 알림 수신 즉시 앱에서 결제 완료 후, 다음 예약에서 자동결제 동작 여부를 재확인하세요.

## 6) 앱 캐시/버전 이슈
- 코레일톡 구버전 또는 캐시 문제로 저장값이 반영되지 않는 사례가 있습니다.
- 앱 업데이트 → 로그아웃/재로그인 → 필요 시 재설치 후 재등록을 권장합니다.

## 7) 반복 실패 시(핵심)
- 등록 카드 삭제 → 앱 재시작 → 카드 재등록 → 기본카드 지정 → 수동 결제 1회 성공 → 자동결제 재시도 순으로 진행하세요.

- [ ] "기본 결제수단"으로 지정되었는가?
- [ ] 자동결제 동의/약관/본인인증이 완료되었는가?
- [ ] 같은 카드로 최근 수동 결제가 성공했는가?
- [ ] 앱 최신 버전인가?

위 체크리스트를 모두 통과했는데도 실패하면, 코레일 고객센터 문의 시 아래 정보를 함께 전달하면 처리 속도가 빨라집니다.
- 예약번호
- 실패 시각
- 열차번호/구간
- 카드사/카드종류(개인/법인/체크)
- 앱 버전(OS 포함)
